# Security (F8TF Lite)

This repository is public and contains only "Lite" materials.

## Do NOT commit
- passwords, API keys, tokens, SSH keys
- .env files
- real database connection strings
- any customer data or confidential documents
- Pro packs (full diagrams, full checklists, bundles)

## Recommended
- use environment variables for secrets
- keep build artifacts out (bin/, obj/, .vs/)
- publish only minimal runnable examples
